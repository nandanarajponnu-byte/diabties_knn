Overview

This project is a machine learning web application that predicts whether a person is likely to have diabetes using the K-Nearest Neighbors (KNN) algorithm. The application is developed with Python, Scikit-learn, and Streamlit for an interactive frontend.

Users can enter medical details through the web interface, and the model predicts the diabetes outcome in real time.

Features
Diabetes prediction using KNN algorithm
Interactive Streamlit user interface
Real-time prediction results
Easy-to-understand and beginner-friendly project
Clean frontend for user input
Tech Stack
Python
Scikit-learn
Pandas
NumPy
Streamlit
Matplotlib / Seaborn
